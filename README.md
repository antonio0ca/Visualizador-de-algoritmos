## 🎯 Sobre

Ferramenta web que **anima passo a passo** o funcionamento de algoritmos, ajudando a entender visualmente como eles operam sobre os dados. Construída com **JavaScript puro** (sem frameworks), HTML e CSS.

## 🚀 Como usar

Não precisa de build nem instalação — é só abrir no navegador:

```bash
# clone o repositório e abra o index.html
# (ou rode um servidor estático simples)
npx serve .
```

Ou acesse direto pelo **GitHub Pages**, se publicado.

## 📁 Estrutura

```
index.html   — estrutura da página
style.css    — estilos e layout
script.js    — lógica da animação dos algoritmos
```
